# Projet Web Sémantique

Projet de web sémantique - 4A IF

## Installation

### Backend

```bash
cd backend
npm install
```

### Frontend

```bash
cd front
npm install
```

## Lancer l'application

### Terminal 1 - Backend

```bash
cd backend
npm start
```

Le serveur backend démarre sur http://localhost:3000

### Terminal 2 - Frontend

```bash
cd front
npm run dev
```

Le frontend démarre sur http://localhost:5173

**Important** : 
- Utilisez deux terminaux différents, un pour le backend et un pour le frontend.
- ATTENTION, certaines pages peuvent prendre un certain temps à charger, notamment la page contenant le graphe.
